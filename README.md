# Sistem-Inventaris
Sistem Inventaris adalah aplikasi untuk mengelola data barang secara mudah dan efisien. Sistem menyediakan fitur login, CRUD data barang, serta transaksi barang masuk dan barang keluar. Dengan sistem ini, stok barang dapat dipantau dan diperbarui secara akurat sehingga memudahkan pengelolaan inventaris.
